## 功能
提供Web学习网站

## 启动
1. 下载所需依赖  
```bash
npm install
```
2. 启动产品模式  
```bash
npm run serve
```
3. 启动开发模式 
```bash
npm run dev
```  
启动开发模式会同时执行多个命令
- `npm run watch-sass` 对Sass进行监视
- `npm run serve` 运行node服务器
